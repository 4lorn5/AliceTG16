Notes on the game's translation process.

## AliceTG16_Eng1.0.ips

The file needed to patch 'Fushigi no Yume no Alice' PC Engine ROM.

## PCE Alice Patcher.py

Small and compact Python patcher (with GUI). Simple to use:

1) ROM File: requires location of the original ROM file.
2) IPS Patch: requires location of the original .IPS file.
3) Output: output folder and file name (editable) for the final, patched version.

The original game ROM must pass the following checks:

File/ROM SHA-1: C266B05DD367B2973399B8DAFF7C4DC33BEBEDD9
File/ROM CRC32: 12C4E6FD
